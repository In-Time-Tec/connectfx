# Connectfx documentation site

The consumer documentation lives in [`apps/docs`](../../apps/docs) and is published at:

- Production: https://connectfx-docs.up.railway.app
- Staging: https://connectfx-docs-staging.up.railway.app

The site is a FoldKit application using FoldCN components generated from the local FoldCN registry. Product contracts remain canonical under [`docs/spec`](../spec), indexed by [`SPEC.md`](../../SPEC.md), with vocabulary in [`CONTEXT.md`](../../CONTEXT.md).
